System Notice (අඩුපාඩු/Shortcomings & Limitations)

  • 1. SMS / WhatsApp Messages: Enabled (WhatsApp) දැන් WhatsApp හරහා පණිවිඩ යැවිය හැක. SMS සඳහා backend (API) සේවාවක් අවශ්‍ය වේ.
  • 2. PDF Generation: Currently mocked. In production, a library like `jsPDF` or a backend printing service is needed to generate formatted certificates/reports.
  • 3. Data Persistence (දත්ත ගබඩා කිරීම): Succeed (දැන් දත්ත ස්වයංක්‍රීයව ගබඩා වේ) Local Storage මඟින් දැන් ඔබේ දත්ත ආරක්ෂිතව ගබඩා කෙරේ.
  • 4. File Uploads: Selecting files works locally but they are not saved to a server permanently across different browsers.
